Example 1

%{
create two random vector A and B then use a for loop to create a vector C
such that:
> first element of C is sum of firstelement of A and last element of B
> and second element of C is sum of second element of A and second
  last element of B
%}
clc; clear;
N = 10;
A = randi(100,N,1);
B = randi(100,N,1);   % creating a B matrix of 10X1
C = zeros(N,1);
for i = 1:N
    C(i,1) = A(i,1)+B(N+1-i,1);
end
disp(A);
disp(B);
disp(C);
